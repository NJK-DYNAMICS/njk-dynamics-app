# NJK Nexus — Portal de Aplicativos

Portal central do ecossistema **NJK Dynamics**.

## Estrutura

| Caminho | Descrição |
|---------|-----------|
| `index.html` | Shell HTML (deploy Vercel) |
| `../../src/pages/nexus.js` | Lógica do portal |
| `../../src/styles/nexus.css` | Estilos específicos |
| `../../src/styles/tokens.css` | Design tokens compartilhados |

## Como executar

Sirva a partir da **raiz do repositório** (Live Server, `npx serve .`, etc.) e abra:

`http://localhost:PORT/apps/nexus/index.html`

**Senha de testes:** `NJK@2025`
